Albums

Uploading Images

  1. Prepare the files. These can be JPG, PNG and non-animated GIF images.
    • Image size should not exceed 2800×1440 pixels.
    • Please use only Latin letters or numbers in the file names. No special symbols are allowed.
    • We recommend creating meaningful filenames — e.g. MyGallery_image01.jpg.
  2. Create a new Album. Open it.
  3. Select the image files to upload from your computer.
  4. Repeat the previous step for all images to be uploaded to the server.
  5. Click “Start Upload” button. All image files will be stored in the directory /images/globalflashgalleries/
